100G Ethernet Ref Design for 2x QSFP28 FMC
==========================================

This is the documentation for the 100G Ethernet reference design for the Opsero `2x QSFP28 FMC`_.
The design uses the Versal Integrated 100G Multirate Ethernet MAC (MRMAC) to drive each QSFP28
port as a single 100GbE (CAUI-4) channel.
